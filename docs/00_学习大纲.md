# Combine 学习大纲

## 什么是 Combine？

Combine 是 Apple 在 WWDC 2019 推出的**响应式编程框架**（iOS 13+）。它提供了一套声明式的 Swift API，用于处理随时间变化的值（如用户输入、网络响应、定时器等）。

### 核心思想

> 数据从**发布者（Publisher）**流向**订阅者（Subscriber）**，中间可以通过**操作符（Operator）**进行转换、过滤、组合。

```
Publisher → Operator → Operator → ... → Subscriber
```

这类似于 Swift 的高阶函数链式调用（map/filter/reduce），但处理的是**异步**事件流而非同步集合。

### 与其他方案的对比

| 方案 | 特点 |
|------|------|
| **Delegate** | 一对一，需要定义协议，代码分散 |
| **Closure/Callback** | 容易嵌套（回调地狱），难以组合 |
| **NotificationCenter** | 松耦合但类型不安全，难以追踪 |
| **RxSwift** | 功能强大但是第三方库，API 庞大 |
| **Combine** | Apple 原生，与 SwiftUI/UIKit 深度集成，类型安全 |

## 学习建议

1. **按顺序推进**：每一章都建立在前面章节的基础上
2. **运行代码**：每个按钮对应一个独立的 Demo，点击后观察日志输出
3. **修改参数**：尝试改变 debounce 时间、retry 次数等参数，观察行为变化
4. **关注 `print()` 操作符**：它能在 Xcode 控制台显示 Combine 管道的完整生命周期
5. **注意内存管理**：留意代码中 `[weak self]` 和 `.store(in: &cancellables)` 的使用
